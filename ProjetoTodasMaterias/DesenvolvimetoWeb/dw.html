Desenvolvimento Web

Na disciplina de desenvolvimento web aprendemos o processo de criar um site e atribuir design a ele e também implementar código para o seu funcionamento.

    Existem 3 especializações para desenvolvedores web:

    Front-end: encarregado da parte visual e interativa do site (HTML, CSS e JavaScript).

    Back-end: É atribuído a ele a tarefa de fazer o site funcionar e da parte lógica (Python, Java,PHP e C#) .

    Full-stack: Domina os dois tipos de conhecimento Front-end e Back-end.

Uma das plataformas que utilizamos foi o Figma, por que ele ajuda a ter uma base do design que vai ficar o site, sendo bom para criar protótipos. Ele é fácil de se mexer e de simples entendimento, por conta da interface intuitiva, além do mais, tem muitos recursos na página que ajuda a fazer um layout interessante para o site e é possível trabalhar em equipe simultaneamente, o tornando uma escolha interessante.

Fundamentos

HTML (HyperText Markup Language) : estrutura o conteúdo no site

CSS (Cascading Style Sheets): Deixa a estrutura do site com estilo

JavaScript: Dá funcionalidade e cria interatividade para o site

Usamos o Visual Studio Code para programarmos os nossos sites. Ele é muito utilizado por desenvolvedores por ser muito versátil e por ter vários recursos. Ele funciona em diferentes sistemas operacionais, suporta várias linguagens e, utilizando extensões, pode suportar mais. Ele também autocompleta os códigos e dá sugestões. É possível instalá-lo no site oficial e ele é gratuito.
